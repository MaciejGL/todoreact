# TODO React App
 Project build on React.

 
 ### To run app
 
 * Clone repositry,
 * Go into directory `cd todoreact`
 * Run `npm install`
 * Run `npm start`
 
 Your app will be available at http://localhost:3000/

## Goal of the project:
Create an app to store Your todos.
